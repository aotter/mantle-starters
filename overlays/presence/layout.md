# Presence layout hint

Start with one homepage:

- hero: brand, short purpose, primary contact action;
- about: a short credibility or positioning paragraph;
- features: plain value points;
- proof: logos, metrics, testimonials, or examples only when useful;
- contact: one clear prompt and the contact form path;
- cta: one final next step.

Use Kiwa primitives already present in the starter. Do not add a custom
theme layer until the content and route shape are stable.
