# `themes/l4-playful-pop`

L4-tier playful-pop theme. v0.0.9 mechanism stub — artist deliverable
pending; the visual leans bright/loose (warm-off-white paper, hot-pink
accent, geometric sans, wider measure).

## Tokens this theme overrides

| Token | Baseline | This theme |
|---|---|---|
| `--paper` | `#ffffff` | `#fffaf0` (warm off-white) |
| `--ink` | `#1a1a1a` | `#1f1f2e` (deep indigo) |
| `--rule` | `#e5e5e5` | `#ffe4d1` |
| `--rule-strong` | `#c0c0c0` | `#ffb88a` |
| `--mute` | `#6b6b6b` | `#6e6e85` |
| `--accent` | `#2563eb` (blue) | `#ff2e88` (hot pink) |
| `--accent-soft` | `#60a5fa` | `#ff7ab0` |
| `--selection` | blue-tinted | hot-pink-tinted |
| `--font-display` | system-ui | Inter / Söhne / sans |
| `--font-body` | system-ui | Inter / Söhne / sans |
| `--measure` | `38rem` | `44rem` (wider — less precious) |

Dark mode adjusted in parallel (deep-indigo paper, lighter hot-pink).
